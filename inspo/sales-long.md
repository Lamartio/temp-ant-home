Long Description / Landing Page

Take Control of Your Organization's Security

License Management puts you in the driver's seat of your enterprise security. Designed for organizations that demand precision access control, this powerful administration tool lets you define exactly who can do what across your entire ANT platform.

Key Features:

- Role-Based Access Control (RBAC) — Create custom roles with granular permissions. From read-only viewers to full administrators, define access levels that match your organizational hierarchy.
- Centralized User Management — Invite team members, assign roles, and manage access from a single dashboard. Revoke access instantly when team members change roles or leave.
- Two-Factor Authentication Enforcement — Strengthen security by requiring 2FA for specific users or roles. Protect sensitive data with an additional layer of verification.
- App & Table Permissions — Control access to applications and data tables at the license level. Ensure teams only see what they need to see.
- Project Template Management — Create and manage reusable project templates. Standardize workflows and accelerate project setup across your organization.
- Trigger & Label Administration — Manage automation triggers and organizational labels. Keep your workflows consistent and well-organized.

Built for Enterprise

Whether you're managing a team of 10 or 10,000, License Management scales with your organization. The intuitive interface makes complex permission structures simple to manage, while comprehensive audit capabilities keep you compliant.

Project Permissions (ant-app-project-rbac)

Short Description (App Store)

Fine-grained access control at the project level. Manage team roles, task permissions, and data access to keep your projects secure and organized.

Long Description / Landing Page

Secure Collaboration Starts Here

Project Permissions gives project managers complete control over who can access what within their projects. Define team roles, set granular permissions, and ensure sensitive project data stays protected — all without involving IT.

Key Features:

- Project-Level RBAC — Create roles tailored to your project's needs. Assign permissions for team members, contractors, and stakeholders with precision.
- Task Management Permissions — Control who can view, create, edit, or delete tasks. Keep your project timeline secure from unauthorized changes.
- Team Member Management — Add project collaborators from your organization's user pool. Remove access instantly when project phases complete or team composition changes.
- Data Table Access Control — Set read, create, update, delete, and configure permissions on project data tables. Protect sensitive project information with row-level security.
- App Integration Control — Manage which project applications each role can access. Keep specialized tools available only to those who need them.
- Structure Breakdown (SBS) Permissions — Control access to your project's organizational hierarchy. Maintain clarity in complex project structures.
- Automation & Label Management — Define who can create and manage project triggers and labels. Keep workflows organized and automation under control.
- Role Templates — Apply pre-configured role templates from your organization. Accelerate project setup with consistent permission structures.

Empower Your Project Teams

Project Permissions puts security decisions where they belong — with the people who understand the project best. Project managers can configure access without waiting for IT support, while organizational policies remain enforced.
